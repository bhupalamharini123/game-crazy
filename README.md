# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**.
The user plays against the computer, and the scores update dynamically after every round.

## ⚙️ How the Game Works

1. The player selects **Rock**, **Paper**, or **Scissors**.
2. The computer randomly generates its choice.
3. The choices are compared using game rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
4. The winner is displayed and the score updates automatically.
